# Selenium Cucumber POM Framework

## Overview

This project is a Selenium Automation Testing Framework developed using Java, Selenium WebDriver, TestNG, Cucumber BDD, and Page Object Model (POM) design pattern.

The framework is designed to provide a scalable, maintainable, and reusable automation solution for web application testing.

## Technologies Used

* Java
* Selenium WebDriver
* Cucumber BDD
* TestNG
* Maven
* Page Object Model (POM)
* Fillo (Excel Data Handling)
* Apache POI
* Git & GitHub

## Framework Structure

* Feature Files
* Step Definitions
* Page Classes
* Utility Classes
* Test Data (Excel)
* Configuration Files
* Test Runner
* Reports & Screenshots

## Features

* Data-Driven Testing using Excel
* Page Object Model Implementation
* Reusable Utility Methods
* Cross Browser Support
* Screenshot Capture
* Cucumber BDD Reporting
* Maven Dependency Management

## Test Scenarios Covered

### Login Module

* Valid Login
* Invalid Login
* Empty Username
* Multiple Credential Validation

### Lead Module

* Create New Lead
* Lead Validation Scenarios

## Project Execution

### Run Using TestNG

```bash
mvn clean test
```

### Run Using Cucumber Test Runner

Execute the TestRunner class from the IDE.
